# Customers
## GET `/customers`
This route is used to get all customers.

- Method: `GET`
- Endpoint: `https://api.lancero.app/customers`

## GET `/customers/email/:email`
This route is used to find information about a customer.

- Method: `GET`
- Endpoint: `https://api.lancero.app/customers/email/:code`

## POST `/customers/create`
This route is used to create a new customer.

- Method: `POST`
- Endpoint: `https://api.lancero.app/customers/create`

### Request body
- firstname: Optional, the first name of the customer
- lastname: Optional, the last name of the customer
- email: The email of the customer
- notes: Optional, any notes you wish to add to this customer
- waitlist: Optional, default `false`, should this user automatically be added to the waitlist?

```json
{
    "firstname": "Dries",
    "lastname": "Augustyns",
    "email": "dries@lancero.app",
    "waitlist": true
}
```
